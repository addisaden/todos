# todos

todos is a small tool to manage nested todolists.

## Installation

Download from http://uploads.23g.eu/clojure/todos-0.1.1-standalone.jar

## Usage

Start the app and type help to get started.

    $ java -jar todos-0.1.0-standalone.jar
